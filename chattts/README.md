# 1.下载安装
```
1.下载源代码
git clone https://github.com/2noise/ChatTTS
2.进入源代码下载的目录
cd ChatTTS
3.创建 chattts 环境
conda create -n chattts
4.激活环境
conda activate chattts
 (如果这一步报错：CondaError: Run 'conda init' before 'conda activate'
 解决方案：
 conda int
 )
5.安装必要的依赖
pip install -r requirements.txt
```
